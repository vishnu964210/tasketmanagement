Task Management System
A Python + MySQL mini project for managing employees, tasks, and reports.
It provides two modes of operation: Admin and Employee, each with their own menus and functionalities.

Features
👨‍💼 Admin Menu
Dashboard

View total employees

View total tasks

Check task status

Employee Management

Add new employees

Update employee details

Delete employees

View employee list

Task Management

Create tasks

Assign tasks to employees

Update task details

Delete tasks

Task Monitoring

View all tasks

Check task status

Track progress

Reports

Task completion report

Employee performance grading

Overdue task report

👩‍💻 Employee Menu
Dashboard → View personal details

My Tasks → See assigned tasks

Task Details → View task information

Update Status → Change task status (e.g., completed, partial)

Comments → Add comments to tasks

🛠️ Tech Stack
Language: Python 3

Database: MySQL

Connector: mysql.connector
